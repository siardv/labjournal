Data

by: Siard van den Bosch

1 Data

To conduct a comprehensive social network analysis, it is essential to gather and describe key data values and concepts. Before proceeding with the analysis, the following elements should be thoroughly evaluated and obtained:

1.1 Nodes

Nodes refer to the individual researchers (authors) that will form the basis of the co-authorship network. It is important to gather their names and unique identifiers.

1.2 Node Attributes

For each researcher, data on their gender identity, academic rank/position, departmental affiliation, and any other relevant attributes should be collected.

1.3 Ties

Ties represent the co-authorship relationships between researchers. It is crucial to systematically capture every instance of co-authorship to obtain comprehensive data on ties.

1.4 Tie Attributes

Details about each co-authorship tie, such as the publication name, publication year, journal name, etc., can provide valuable attributes for analyzing the ties.

1.5 Network Boundary

Clearly defining the criteria used to delineate the network boundary is essential. This involves determining which researchers and co-authorship ties are included or excluded from the faculty co-authorship network.

1.6 Temporal Data

Capturing information about the time periods when co-authorship ties were active enables temporal network analysis and the study of network evolution.

1.7 Missing Data

Documenting any issues with incomplete data on nodes, ties, attributes, etc., is important. Missing data can introduce biases in network metrics and analysis.

1.8 Data Structure

The format and structure of the network data must be compatible with the social network analysis software tools that will be used. This may require restructuring the data.

Careful evaluation and pre-processing of these data elements are crucial for enabling robust analysis and accurate interpretation of the results. It is essential to thoroughly document the data collection and preparation stage before proceeding with the application of network metrics and models.
