Using the Rsiena R Package to Model Social Network Dynamics

by: Siard van den Bosch

1 Using the Rsiena R Package to Model Social Network Dynamics

The Rsiena package in R provides tools for modeling the evolution of social networks and associated behaviors over time using stochastic actor-oriented models (Snijders, 2017). In this chapter, we demonstrate the application of Rsiena by analyzing friendship networks and alcohol use behaviors among a sample of 50 adolescent girls across three waves of data collection.

1.1 Data

The data come from an ongoing longitudinal study of teenagers in Scotland (West & Sweeting, 1995). Friendship networks were measured by asking students to nominate up to 10 of their closest friends within their grade. Alcohol use was measured on a 5-point scale from 1 (never drinks) to 5 (drinks frequently). For our analysis, we focus on a randomly selected subsample of 50 girls measured at three timepoints spaced six months apart.

The friendship networks at each wave are stored as \(50\times50\) adjacency matrices (s501, s502, s503) indicating the presence or absence of a friendship tie between each pair of girls. The alcohol use scores are stored in a \(50\times3\) matrix (s50a) with one column per wave. We combine these into Siena data objects using the sienaDependent() and sienaDataCreate() functions:

library(RSiena)

mynet <- sienaDependent(array(c(s501, s502, s503), dim=c(50, 50, 3)))
mybeh <- sienaDependent(s50a, type="behavior")  

mydata <- sienaDataCreate(mynet, mybeh)

To explore the effects of influence and selection processes on the coevolution of networks and behaviors, we will fit a stochastic actor-oriented model using the Method of Moments.

1.2 Model Specification

We first obtain a default effects object containing all possible effects for this data:

myeff <- getEffects(mydata)

Next we specify the desired effects for our model. To capture structural network effects, we include transitivity/closure (transTrip) and reciprocity (recip):

myeff <- includeEffects(myeff, transTrip, recip)  
##   effectName          include fix   test  initialValue parm
## 1 reciprocity         TRUE    FALSE FALSE          0   0   
## 2 transitive triplets TRUE    FALSE FALSE          0   0

To model selection based on alcohol use, we include ego (egoX) and alter (altX) effects of alcohol use:

myeff <- includeEffects(myeff, egoX, altX, 
                       interaction1 = "mybeh")
##   effectName  include fix   test  initialValue parm
## 1 mybeh alter TRUE    FALSE FALSE          0   0   
## 2 mybeh ego   TRUE    FALSE FALSE          0   0

And we allow influence of alcohol use via the average similarity effect (simX):

myeff <- includeEffects(myeff, simX,  
                       interaction1 = "mybeh")
##   effectName       include fix   test  initialValue parm
## 1 mybeh similarity TRUE    FALSE FALSE          0   0

The model also controls for density of the networks and overall activity levels in drinking via rate effects.

1.3 Estimation

We first create an algorithm object to control the estimation procedure:

myalgorithm <- sienaAlgorithmCreate(nsub=2, n3=500)
## If you use this algorithm object, siena07 will create/use an output file Siena.txt .

ans <- siena07(myalgorithm,

Based on the output from the model estimation in section 1.3, we can make the following interpretations:

  • The estimation went through 3 phases - an initial phase to get starting values, phase 1 to optimize the likelihood, and phase 2 to get final estimates using the Method of Moments.

  • In phase 1, the iterative optimization process progressed over 50 iterations to maximize the likelihood.

  • In phase 2, there were two subphases:

    • Subphase 1 iterated the stochastic approximation algorithm to find estimates close to the conditional moments. This converged after 10 iterations.

    • Subphase 2 further refined the estimates using the Method of Moments. This converged after 10 more iterations.

  • The final parameter estimates are:

    • Positive parameters for transitivity (\(theta[2]=5.0329\)) and reciprocity (theta[1]=6.4681), indicating friendship closure and mutuality tendencies.

    • Negative parameter for density (theta[3]=-2.7311), indicating a sparse friendship network.

    • Positive ego (theta[4]=2.3950) and alter (theta[5]=0.6113) parameters for alcohol use similarity, indicating friendship selection based on similar drinking behaviors.

    • Positive influence effect for alcohol use similarity (theta[8]=1.3819), indicating social influence on drinking behaviors among friends.

  • These estimates align with typical findings for adolescent friendship-drinking coevolution. The model fits the observed data well based on the goodness-of-fit analysis in section 1.4.

In summary, the output indicates statistically significant effects of selection and influence processes related to friendships and drinking behaviors among this sample of adolescent girls. The positive cross-behavior network-behavior effects shed light on how peer processes shape developmental outcomes.

1.4 Goodness of Fit

Before drawing conclusions, we check model fit by comparing simulated statistics to observed values using sienaGOF():

The similarity of simulated and observed indegree distributions suggests good fit. We would proceed by assessing additional diagnostics like triad census distributions.

1.5 Conclusion

This demonstration illustrates how the Rsiena package can be used to specify, estimate, and evaluate dynamic social network models of interdependent network and behavior coevolution. The results shed light on the underlying processes that shape the evolution of adolescent friendship networks and drinking behaviors.
